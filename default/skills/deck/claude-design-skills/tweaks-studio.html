Design reference · v1

A tweakable surface for everything on this page.

Toggle Tweaks in the toolbar, then click any element to inspect and change it. Drop comments anywhere. Everything persists back into the file.

Tokens

Fonts, colors, spacing, radii, shadows — all live in a single JSON block.

Picking

Hover to hint, click to lock. The sidebar shows that element's properties.

Comments

Pins, element flags, and draggable sticky notes. All anchored and persisted.

Why bake the editor into the page?

A design file shouldn't need a separate editor app. When the tokens, the text, and the feedback all live in the same HTML, iteration gets trivial — reload, everything is where you left it.

Double-click any paragraph to edit it in place. Or pick it, and edit in the sidebar. Either way, the new copy ends up in the JSON block at the bottom of the file.
